## Session 2026-05-30-T002-attempt-1

### Scenario 1 — Write round-trips through the existing read path

Proves that a pointer written by `write_persisted_default(S, R)` is read
back by the existing `resolve_repository_root_with(..., Some(pointer))`
source-3 path and resolves to the canonical repo `R`, and that
`persisted_default_present(S)` flips to `true`.

<red>
Before this task, `patina-core::discovery::repo` had no public write or
existence surface — only the private `read_persisted_default` /
`persisted_default_repo_path` helpers. A test exercising
`write_persisted_default` / `default_repo_pointer_path` /
`persisted_default_present` does not compile:

```
error[E0432]: unresolved import `patina_core::discovery::write_persisted_default`

  |

```
</red>

<green>
With the writer landed:


</green>
