### Scenario 4 — A `TemplateRender` file entry emits no `mode` key (task scenario 4)

Test: `config::writer::tests::append_template_render_omits_mode_key`.

<red>
`mode_manifest_str` returns `None` for `FileMode::TemplateRender`, so
`append_file_entry` inserts no `mode` key. A regression that made the
template arm emit any string would add a `mode = ` line and fail the

sensible alternate string exists — the parser rejects an explicit
